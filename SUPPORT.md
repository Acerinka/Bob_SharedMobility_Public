# Support

Use GitHub Issues for reproducible project problems and GitHub Discussions or direct maintainer contact for broader questions if enabled.

Good support requests include:

- Unity version.
- Platform and hardware.
- Scene name.
- Exact input sequence.
- Console logs.
- Screenshots or screen recordings when the issue is visual.
- Whether the issue reproduces from a clean checkout of `main`.

For delivery verification, start with `docs/DELIVERY_REGRESSION_CHECKLIST.md`.
