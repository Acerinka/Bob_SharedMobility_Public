# Code Of Conduct

This repository is maintained as a professional collaboration space.

## Standards

- Be respectful and specific.
- Critique code, behavior, and project risk rather than people.
- Assume good intent while still naming concrete problems.
- Keep review feedback actionable.
- Avoid harassment, insults, discriminatory language, or personal attacks.

## Reporting

If you see behavior that makes the project unsafe or unproductive, contact the repository owner through GitHub. Maintainers may remove comments, close issues, or restrict access when conduct harms the collaboration.
